# 🚀 TaskFlow Pro – To-Do Web App

A modern, feature-rich To-Do List web application built using HTML, CSS, JavaScript, jQuery, and Bootstrap.

## 🌐 Live Demo
👉 (https://github.com/varshithakrishnappa/TODO-App.git)

---

## ✨ Features

- ✅ Add tasks (button + Enter key)
- ✏️ Edit tasks
- ❌ Delete tasks
- ✔️ Mark tasks as completed
- 📅 Add due dates
- 🔍 Search tasks
- 📊 Task statistics (Total / Completed / Pending)
- 🧲 Drag & Drop task reordering
- 🌙 Dark mode toggle
- 💾 Data persistence using localStorage
- 🖼️ Empty state illustration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- jQuery
- SortableJS (for drag & drop)
- LocalStorage API
